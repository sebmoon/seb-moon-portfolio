## Overview

Designed a ball sorting and transport system using LEGO SPIKE. Two buggies
were constructed: one transporting balls from the input location to the
sorting area, the other from the sorting area to their respective output
locations. The system included a tipping mechanism to move balls from the
input to the first buggy, an organising wheel to separate balls by colour,
a colour sensor to detect each ball's colour, and a sorting arm to pick up
and move each ball to its output location based on colour — together, a
reliable and efficient sorting and transport system.
