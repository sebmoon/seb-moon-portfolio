summary: >-
  Designed a ball sorting and transport system using LEGO SPIKE. To sort and
  transport the balls, two buggies were constructed. One buggy was used to
  transport the balls from the input location to the sorting area, while the
  other buggy was used to transport the balls from the sorting area to their
  respective output locations. The system also included a tipping mechanism to
  move the balls from the input location to the first buggy, an organising wheel
  to separate the balls by colour, and a colour sensor to detect the colour of
  each ball. Finally, a sorting arm was used to pick up and move each ball to
  its respective output location based on its colour. With these components, a
  reliable and efficient ball sorting and transport system was created using
  LEGO SPIKE.
